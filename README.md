# My React App

This is a React frontend project built with **Vite**. It includes multiple pages with basic styling and routing.

---

## Prerequisites

- [Node.js](https://nodejs.org/) (v16+ recommended)  
- npm (comes with Node.js) or yarn  

---

## How to Run the Project

### 1. Clone the repository

```bash
git clone git@gitlab.rn.inf.tu-dresden.de:scc/scc-gruppe-08/tripduel/frontend.git

cd my-react-app
```

### 2. Install dependencies
```bash
npm install
```

Or, if you use yarn:

```bash
yarn
```

### 3. Start the development server

```bash
npm run dev
```

Open your browser at http://localhost:5173


The app supports hot reload, so changes in the src/ folder are reflected automatically in the browser.
